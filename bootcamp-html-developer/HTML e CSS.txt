Curso de HTML e CSS

Tags:

Destaque para títulos e conteúdos relevantes: <h1> até <h6>.

Particionamento de conteúdo:
<header> -> Cabeçalho da página ou de outra seção, como section, article ou aside por exemplo.
<section> -> Nova partição dentro do conteúdo.
<footer> -> Rodapé da página ou de outro conteúdo, como section ou article por exemplo.
<article> -> Conteúdo a ser incluído em outro principal.
<aside> -> Conteúdo secundário como barra lateral por exemplo.

Tags para textos:
<p> -> Parágrafo para textos mais densos onde os "h's" seriam inviáveis.

Tag para link:
<a> -> (âncora), tag utilizado para interligar conteúdos na web. Pode ser utilizado para redirecionamento de sites, email ou telefone.
<a href="site.com/pagina">"link a ser exibido"</a>
<a href="mailto:email@servidor.com">E-mail</a>
<a href="tel:159XXXXXXXX">Ligue</a>

Para que o link abra em nova aba, deve-se adicionar o target="_blank".
<a href="link" target="_blank">Nome a ser exibido para link</a>
